# LARAVEL

## Persyaratan sistem

PHP minimal 8.0

sudah terinstall composer
<https://getcomposer.org/>

sudah terinstall nodejs
<https://nodejs.org/>

## instalasi

```bash
cp .env.example .env
```

```bash
composer install
```

```bash
npm install
```

```bash
npm run build
```

```bash
php artisan migrate --seed
```

```bash
php artisan key:generate
```

```bash
php artisan serve
```

buka <http://127.0.0.1:8000/>
